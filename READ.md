# 🛒 SuperMarket Sales Analysis

## 📌 Project Overview

This project was completed as part of my Data Analytics Internship.

The objective of this project is to analyze supermarket sales data using Python and perform Exploratory Data Analysis (EDA) to generate meaningful business insights.
